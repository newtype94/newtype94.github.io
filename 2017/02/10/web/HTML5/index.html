input(입력양식) 태그

<input type=""> 속성(type)이 성격을 결정함

  1. 직접 입력하는 타입
    <input type="text" name="액션페이지로 전달매개" value="전달값(디폴트값)" />

    type 기능
    text 텍스트상자
    password 암호화된 텍스트상자
    file 파일선택버튼
    checkbox 체크박스
    radio 라디오버튼

  2. 버튼을 누르는 타입
    <input type="button" value="버튼이름" />

    type 기능
    button 버튼
    reset 리셋버튼
    submit 제출버튼
    image 이미지

  3. 안 보이는 타입
    <input type="hidden" name="hidden" value="hidden" />

    type 기능
    hidden 안보임
